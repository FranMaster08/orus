<strong>Stack:<strong> NodeJS | NestJS | TypeScript | MySQL
  




## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# instalacion global de nest
$ npm install -g @nest/cli

# crear nuevo proyecto nestjs
$ nest new project <name>

# crear nuevo controller
$ nest g controller <name>

# crear nuevo servicio
$ nest g service <name>


```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
